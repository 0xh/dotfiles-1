## How To Use

``` bash
git clone https://github.com/wfxr/dotfiles
cd dotfiles
```

- Restore specified configuration(s):

``` bash
  # eg: restore zsh, tmux, vim and git
  ./install_config.sh zsh tmux vim git
```
